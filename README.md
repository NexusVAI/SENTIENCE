## рҹҡҖ Installation

### Requirements
- GTA 5 (Steam/Epic/Rockstar)
- [ScriptHookV](http://www.dev-c.com/gtav/scripthookv/)
- [ScriptHookVDotNet 3](https://github.com/scripthookvdotnet/scripthookvdotnet)
- One of:
  - [LM Studio](https://lmstudio.ai/) (free, local AI)
  - DeepSeek API key (cheap, cloud AI)
  - Any OpenAI-compatible API

### Quick Start (Cloud вҖ” 2 minutes)

```bash
1. Install ScriptHookV + ScriptHookVDotNet
2. Copy GTA5MOD2026.dll + Newtonsoft.Json.dll вҶ’ GTA5/scripts/
3. Launch GTA 5 once (creates config file)
4. Edit: Documents/GTA5MOD2026/config.ini
     Provider=cloud
     CloudEndpoint=https://api.deepseek.com/v1/chat/completions
     CloudModel=deepseek-chat
     CloudAPIKey=sk-your-key-here
5. Restart GTA 5
6. Walk up to any NPC. Press G.
7. Watch them come alive.
```

### Free Setup (Local AI)

```bash
1. Install LM Studio вҶ’ download a model:
     Weak PC:   qwen2.5-1.5b-instruct
     Mid PC:    qwen2.5-3b-instruct
     Strong PC: qwen3.5-4b
2. Start LM Studio server (port 1234)
3. Edit config.ini:
     Provider=local
4. Launch GTA 5
```

## вҡҷпёҸ Configuration

All settings in `Documents/GTA5MOD2026/config.ini`:

```ini
[LLM]
# local = LM Studio, cloud = API
Provider=local
LocalEndpoint=http://127.0.0.1:1234/v1/chat/completions
LocalModel=qwen2.5-3b-instruct
CloudEndpoint=https://api.deepseek.com/v1/chat/completions
CloudModel=deepseek-chat
CloudAPIKey=

[Performance]
MaxTokens=80
Temperature=0.7
MaxDialogueLength=20
RequestCooldown=2
# StrictMode: instant responses for simple interactions
StrictMode=True

[TTS]
TTSProvider=edge
VoiceEnabled=true

[Awakening]
Enabled=true
# 1=slow 2=normal 5=fast
Speed=2
```

### Supported AI Providers

| Provider | Endpoint | Model | Cost |
|:--------:|:--------:|:-----:|:----:|

| OpenAI | api.openai.com | gpt-4o-mini | ~$0.01/chat |
| SiliconFlow | api.siliconflow.cn | Various | Cheap |
| Any OpenAI-compatible | Custom URL | Custom | Varies |
