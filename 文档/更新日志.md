\## v3.1 (2026-03-01)

\### 新增

\- 配置文件系统 ( ModConfig.cs ) ：

\- 实现了完整的 INI 配置文件读取和保存逻辑。

\- 支持 \[LLM] (Provider, Endpoint, Model, APIKey), \[Performance] (MaxTokens, Temperature), \[TTS] , \[Awakening] 等配置项。

\- 首次运行时会自动生成 Documents/GTA5MOD2026/config.ini 。

\- AI 管理器更新 ( AIManager.cs ) ：

\- 现在从 ModConfig 加载配置。

\- 支持云端 API（DeepSeek/OpenAI），自动处理 Authorization 头。

\- 支持本地 LLM (LM Studio)。

\- 实现了基于配置的智能对话截断 ( MaxDialogueLength )。

\- 语音管理器重构 ( VoiceManager.cs ) ：

\- 完全重写，支持 ModConfig 。

\- 实现了 Python edge-tts 进程预热 ( PreWarmEdgeTTS )，消除首次对话延迟。

* 实现了 TTS 服务器优先 + CLI 回退机制。

\- 优化了 SSML 参数（语速、音调），根据情绪（Angry, Scared, Happy 等）动态调整，声音更自然。

\- 使用 NAudio 替代 PowerShell 进行音频播放，大幅降低延迟。

\- NPC 管理器集成 ( NPCManager.cs ) ：

\- 集成 ModConfig ，在 SendInteraction 中使用配置的 MaxTokens 和 Temperature 。

\- 集成 AwakenSystem ，注入觉醒上下文。



\## v3.0 (2026-02-27)
