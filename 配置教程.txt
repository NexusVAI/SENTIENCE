1. 配置文件位置 配置文件 config.ini 必须 放在以下路径： C:\Users\你的用户名\Documents\GTA5MOD2026\config.ini (即：我的文档 -> GTA5MOD2026 -> config.ini)
不是 放在游戏根目录或 scripts 文件夹里。

2. 切换云端 API (DeepSeek/OpenAI) 仅仅填入 API Key 是不够的，需要修改两个地方：
打开 config.ini ，找到 [LLM] 部分：
1. 修改 Provider ：把 Provider=local 改为 Provider=cloud 。
2. 填入 Key ：在 CloudAPIKey= 后面填入你的密钥 (例如 sk-xxxx )。
示例：

[LLM]
Provider=cloud
LocalEndpoint=... (不用管)
LocalModel=... (不用管)
CloudEndpoint=https://api.deepseek.com/v1/chat/
completions
CloudModel=deepseek-chat
CloudAPIKey=sk-你的DeepSeek密钥

原理说明： 代码逻辑 ( AIManager.cs ) 是这样的：

- 如果 Provider 是 "cloud"，它才会去读取 CloudEndpoint 和 CloudAPIKey 。
- 如果 Provider 是 "local" (默认)，它会忽略 Key，直接去请求本地的 LM Studio ( LocalEndpoint )。
所以 务必记得改 Provider ！修改保存后， 重启游戏 即可生效。


例如
# 切换到DeepSeek（1秒响应）：
[LLM]
Provider=cloud
CloudEndpoint=https://api.deepseek.com/v1/chat/completions
CloudModel=deepseek-chat
CloudAPIKey=sk-xxxxx

# 电脑很弱（GT730/核显，4GB内存）：
[LLM]
Provider=local
LocalModel=qwen2.5-1.5b-instruct

# 电脑一般（GT1050/RX570，8GB内存）：
[LLM]
Provider=local
LocalModel=qwen2.5-3b-instruct

# 电脑较强（RTX2060+，16GB内存）：
[LLM]
Provider=local
LocalModel=qwen2.5-7b-instruct

# 不想折腾（需要网络，每次调用花几分钱）：
[LLM]
Provider=cloud
CloudModel=deepseek-chat

# 切换到OpenAI：
[LLM]
Provider=cloud
CloudEndpoint=https://api.openai.com/v1/chat/completions
CloudModel=gpt-4o-mini
CloudAPIKey=sk-xxxxx

# 切换到任何兼容OpenAI的API：
[LLM]
Provider=cloud
CloudEndpoint=https://whatever-api.com/v1/chat/completions
CloudModel=whatever-model
CloudAPIKey=whatever-key
