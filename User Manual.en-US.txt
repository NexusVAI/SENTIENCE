# AI NPCS Installer – User Guide

## 1. Overview

The AI NPCS Installer provides a seamless way to integrate advanced AI-driven NPCs into **GTA V**. Designed for both ease of use and reliability, it automates the installation process, validates your environment, and ensures that all required components are correctly deployed.

**Key Features:**

* Automatic detection of GTA V installation directories (Steam, Epic, Rockstar)
* One-click installation of core MOD files with automatic rollback on failure
* Environment verification including Python, dependencies, Whisper models, ScriptHook, and game file integrity
* Start/stop control for the voice server
* Light and dark theme support
* Official website: [https://nexusvai.github.io/NexusV/](https://nexusvai.github.io/NexusV/)

---

## 2. Quick Start

Follow these steps to install AI NPCS and ensure your game environment is ready:

1. Launch `AI_NPCS.exe`.
2. Click **Detect GTA V** to automatically locate your game installation.
3. Click **Check Environment** to verify that all dependencies are installed and correctly configured.
4. Click **Install MOD** to deploy all required files automatically.
5. Click **Full Test** to perform a comprehensive validation of the installation.

---

## 3. Installed Files

After a successful installation, the following files will be deployed:

**Script Directory:**

* `GTA5MOD2026.dll`
* `Newtonsoft.Json.dll`
* `NAudio.dll`

**GTA Root Directory:**

* `ScriptHookVDotNet.asi`
* `ScriptHookVDotNet2.dll`
* `ScriptHookVDotNet3.dll`

**User Configuration:**

* `%USERPROFILE%\Documents\GTA5MOD2026\config.ini`

---

## 4. Troubleshooting

If any issues occur during installation, the installer will automatically revert changes to preserve the integrity of your game files.

**Common Issues:**

* **Missing ScriptHookV:** Ensure `ScriptHookV.dll` and `dinput8.dll` are present in the GTA root directory.
* **Voice server not functioning:** Confirm that Python and required dependencies are installed, and download the Whisper model as needed.

For further assistance, consult the official website or community forums.

---

## 5. Updates

The AI NPCS Installer supports both online and offline updates:

* **Offline Update:** Replace the installer folder with the latest release.
* **Online Update:** Incremental updates are applied, replacing only modified files.

**Recommendation:** Always back up your configuration file located at `Documents\GTA5MOD2026\config.ini` before performing updates.
