# Installation Guide for SENTIENCE

## Prerequisites
- Ensure you have the following installed:
  - [Node.js](https://nodejs.org/en/) (version 14 or above)
  - [Git](https://git-scm.com/)

## Clone the Repository
To get started, clone the repository to your local machine:
```bash
git clone https://github.com/NexusVAI/SENTIENCE.git
```

## Navigate to the Project Directory
Change your directory to the project folder:
```bash
cd SENTIENCE
```

## Install Dependencies
Before running the application, make sure to install the required dependencies:
```bash
npm install
```

## Running the Application
After the dependencies have been installed, you can start the application:
```bash
npm start
```

## Additional Configuration
Depending on your environment, you may need to set up environment variables. Create a `.env` file in your root directory:
```bash
touch .env
```

Fill it with the necessary configurations. Refer to `.env.example` for more details.
